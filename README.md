## Run Locally

Clone the project

```bash
  git@github.com:JuanFelipeObregonRodriguez/react-vet.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
